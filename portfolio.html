StudyBuddy - AI学習支援アプリ

Next.js + FastAPI + PostgreSQL で開発したAI搭載学習管理アプリ。機械学習アルゴリズムでユーザーの学習パターンを分析し、パーソナライズされた学習計画を自動生成。GitHub Stars 500+を獲得した人気プロジェクト。

Next.js FastAPI PostgreSQL TensorFlow Docker

EcoTracker - 環境データ可視化プラットフォーム

React + D3.js + WebSocket で開発したリアルタイム環境データ可視化プラットフォーム。大気質、気温、湿度などの環境データを美しいインタラクティブなグラフで表示。環境省のオープンデータAPIと連携。

React D3.js WebSocket Chart.js Material-UI

SmartVision - リアルタイム画像認識システム

PyTorch + OpenCV + FastAPI で開発したリアルタイム画像認識システム。独自のCNNモデルで物体検出・分類を行い、Webカメラからの映像をリアルタイムで解析。精度92%を達成し、大学の研究室で実用化検討中。

PyTorch OpenCV FastAPI NumPy Streamlit

FitTracker - フィットネス管理アプリ

React Native + Firebase + HealthKit で開発したフィットネス管理アプリ。運動記録、カロリー計算、目標設定機能を搭載。Apple HealthKitと連携してヘルスデータを自動同期。App Store Connect準備中。

React Native Expo Firebase HealthKit Redux

Next.js + Socket.io + Redis で開発したリアルタイムコード共有・編集プラットフォーム。複数人でのリアルタイム編集、音声チャット、画面共有機能を搭載。WebRTCを活用したP2P通信で低遅延を実現。
