I designed my portfolio website using Hugo and published the site on GitHub Pages.
This website was created using the Hugo theme called [Lynx](https://github.com/jpanther/lynx). The process of building up the website so that it could be published on GitHub Pages was difficult because of an error, but I was happy to finally resolve it.

I wanted a website where I could post all of my contact information for Twitter, Instagram, websites, etc. so I created it.
