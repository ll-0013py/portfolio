I designed my portfolio website using Hugo and published the site on GitHub Pages.
This website was created using the Hugo theme called [Developer Portfolio](https://github.com/samrobbins85/hugo-developer-portfolio). It also uses the UIKit framework inside, however I did not change any settings myself.

It was created primarily for the purpose of recording the following information:
- An overview of my personal background and skills
- A dedicated page with my contact information
- A portfolio showcasing my work
- A blog

Here is the link to [My Portfolio Website](https://ll-0013py.github.io/portfolio/).

The sections of this website are as follows:
- Home
- About
- Blog
- Portfolio
- Contact
